About Me

I am Soumyadyuti Dey, a Cloud Architect, AI/ML Engineer, and Cybersecurity professional currently completing my BCA at The NorthCap University. With solid expertise in cloud-native technologies, DevOps, AWS, machine learning, NLP/LLMs, and cybersecurity, I specialize in designing scalable cloud infrastructures and intelligent systems.

My technical journey spans across cloud architecture (AWS, Azure, GCP), AI/ML engineering with focus on intelligent pipelines and LLM-based tools, and cybersecurity implementation including threat simulation and incident response. I've successfully built projects like "On Beat" wellness web app, "Scribe" AI-powered note assistant, "AVAS" advanced voice assistant, and "SentraCore" cybersecurity desktop bot for real-time threat management.

I've gained practical experience as a System Engineer intern at Thinkdex, where I improved AWS deployment, WebRTC latency, and backend efficiency. My achievements include winning multiple hackathons (National-level, NCU Hackutsav, SGT Innoverse) with prizes totaling ₹65,000, and securing certifications in AWS Educate "Introduction to Cloud 101" and DeepLearning.AI + AWS "Generative AI with Large Language Models."

I maintain an active presence on GitHub showcasing cloud architecture demos, ML pipelines, and DevOps setups, and share Postman public API collections. My approach combines academic rigor, cloud-native deployment skills, and hands-on AI/ML toolkit to build secure, scalable systems that drive innovation in the emerging AI-cloud-security space.
